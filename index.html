ChoraleBricks¶

One core functionality of this package is to act as an object-relational mapper for the ChoaraleBricks dataset, meaning that all data entities are linked and represented as classes. Furthermore, it offers helpers to easily sample data from this dataset and provides additional annotations such as F0 trajectories or sheet music alignments.

Citing ChoraleBricks:

If you want to cite ChoraleBricks in a scholarly work, please cite the following article:

1

Stefan Balke, Axel Berndt, and Meinard Müller. “ChoraleBricks: A Modular Multitrack Dataset for Wind Music Research” Under review. 2025.
